### 《电话线》

你说，跪下。我就
跪下了，等黑暗中一闪而过的
耳光
几乎要打瞌睡了
清醒的时候，你也许是好的
给我糖吃，拿玻璃罐里崭新的钞票
炫耀，说我多么幸运啊
被祝福的孩子，几乎是个王子
我可以和你妻子笑得一样好
假使我不曾继承你的话，继承你的丑
对比她洁白的脸庞
和她复制的影子（她们就像挨着的牙齿）
白让你惭愧吗？你洪水般浩盛的
屈辱，转而流向了我
我难道是容忍你的海吗
容忍你妻子的失眠
为她不贞的丈夫，她小女儿的父。
多年以后，你在电话线里 炫耀
你玻璃橱窗的反光
正彰显着
你作为诗人和男人的奖
我真想给你一耳光，你却说：原谅
我这一生的失败
我在你身上学会的暴力
在你的求饶面前 熄灭了
我又成为了
女人，一个跪在
长长走廊里的
王子
可你却不是王

---

- 我想先讨论这首的风格，目前整理了一些思路。整体上，这首诗语言是「去材质」的，所谓「材质」，我定义为更精确的属性信息：颜色、光影、触感、类型、具体功能、使用程度、品牌、装饰物……可以将意象想象成 3D 建模，去掉其表面的材质图层，意象还原为纯的形状。你的语汇多在表格中的「无材质」这列，关乎风格无关优劣。「无材质」会使意象有更多的隐喻的可能性，此诗有之，但读着有些平，可能是因为诗歌需要某种形式的信息量（尤其是新的信息量），如果不在语汇的挑选中，那么就在语汇的组合中，抒情和叙述都是组合的一种。

| 非材质 | 无材质  | 多材质          |
| --- | ---- | ------------ |
| 甜   | 糖    | 硬糖、棒棒糖       |
| 钱   | 钞票   | 崭新的钞票、崭新的人民币 |
| 排列  | 牙齿   | 门牙、智齿        |
| 容忍  | 海    | 深海、地中海       |
| 开心  | 笑    | 苦笑、哂笑        |
| 通道  | 长长走廊 | 学校走廊、走廊右侧    |
| 痛苦  | 失眠   | 半月的失眠、失眠药物   |

下面是一些具体问题：

- 语汇的挑选问题：「王」和「王子」这组比喻对父女关系转变的表达是否足够准确，是否同义反复又构成干扰？
- 新的信息量的问题：「耳光」是常见的暴力动作（在生活中、在写作中都常见），在此诗中「耳光」是实指还是虚指？去掉耳光这首诗似乎也成立，说明对「耳光」没有新信息的生成。反例是「电话线」，即使只出现一次，但构成了核心信息。
- 叙述中信息省略的问题：「我可以和你妻子笑得一样好」，这一句感觉前后省略了信息，「笑」如果是一种家庭营业的话，其原因也是因情况而异的，「笑」的原因是什么？
- 信息量过度的问题：「你作为诗人和男人的奖」，「诗人」这一身份感觉是突然而来的信息量，前后无对应，导致「诗人」即使在事实上成立，在诗里却看不到成立之处。
