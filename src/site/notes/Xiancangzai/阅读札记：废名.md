#### [《灯》](zotero://open-pdf/library/items/Y3SUCQ8D?page=22&annotation=PDBMFYWE)

![Pasted image 20260211173009.png](/img/user/%E9%99%84%E4%BB%B6/attachment/Pasted%20image%2020260211173009.png)

废名的月地爬虫、黾神比纪弦的爬虫篇更科幻。「黾神」乃是自造词。

![Pasted image 20260211173424.png](/img/user/%E9%99%84%E4%BB%B6/attachment/Pasted%20image%2020260211173424.png)

废名咬住了女子作为诗歌工具的坏果子。

![Pasted image 20260211201247.png](/img/user/%E9%99%84%E4%BB%B6/attachment/Pasted%20image%2020260211201247.png)

「灯」：既旧又新的意象。废名的写法是在用意象一次性地博取诗意，是「初学者」的常规动作，这里的「初学者」是指：中国新诗以废名为「初学者」。废名不是老人写新诗，而是新人写新诗。

### [《泪落》](zotero://open-pdf/library/items/Y3SUCQ8D?page=24&annotation=WTZLYP8B)

![Pasted image 20260211202100.png](/img/user/%E9%99%84%E4%BB%B6/attachment/Pasted%20image%2020260211202100.png)

废名的「虎」是偈语式的，但却是低级偈语（这是所有禅诗的堕落）。尽管「堕泪于是……」句几乎不通（废名多有此类浓缩句子），但废名其实更接近后来的某些口语诗，这是废名的好处。

### [《镜》](zotero://open-pdf/library/items/Y3SUCQ8D?page=26&annotation=FBY6FW4Y)

![Pasted image 20260211202659.png](/img/user/%E9%99%84%E4%BB%B6/attachment/Pasted%20image%2020260211202659.png)
